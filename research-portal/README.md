# 📊 AI Research Portal — Financial Statement Extraction Tool

## Overview

This project implements a minimal internal research portal where AI-powered tools produce structured, analyst-ready outputs from uploaded documents.

The current implementation focuses on:

✅ Financial Statement Extraction to Excel  
✅ OCR support for scanned PDFs  
✅ Structured table extraction  
✅ Interactive dashboard UI  
✅ Excel export for analysts  

The system is designed to simulate enterprise research workflows rather than a generic chatbot interface.

---

## 🧠 Key Features

- Upload financial documents (PDF)
- Automatic document ingestion
- OCR fallback for scanned files
- Financial table detection and extraction
- Structured Excel generation
- Dashboard preview of extracted data
- Downloadable Excel output
- Modular backend architecture
